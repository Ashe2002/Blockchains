# Blockchains
This repository contains code for a sealed-bid Vickrey auction for an NFT, to be deployed onto the Solidity Blockchain. The corresponding report evaluates the contract's effectiveness, security, gas efficiency, and user experience. It also
examines a similar auction created by a classmate for comparison, highlighting different approaches to implementation and optimisation.

This was completed as a part of the "Blockchains and Distributed Ledgers" course at the University of Edinburgh.
